# Social Research Institute Themes


How to Install
----------------

```r
devtools::install_github("kmwest/SRI")
```

### Usage
library(SRI)

### Available functions

* theme_sri
* sri_palettes

### Shortcut

Add +sribasic to your ggplot to add a quick basic format. This applies theme_sri and sets scale color and fill to "sricolors"

### See all palettes

name(sri_palettes)
